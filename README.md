# Screenshot App

Diese Screenshot-App ermöglicht es Ihnen, automatisch Screenshots Ihres Bildschirms zu erstellen und in einem angegebenen Ordner zu speichern. Die App wurde mit Python und PyQt5 erstellt und bietet eine einfache Benutzeroberfläche zum Starten und Stoppen der Screenshot-Aufnahme.

## Voraussetzungen

Stellen Sie sicher, dass Sie die folgenden Python-Bibliotheken installiert haben:

- PyQt5
- pyautogui
- Pillow

Sie können diese Bibliotheken mit pip installieren:

```sh
pip install PyQt5 pyautogui Pillow
